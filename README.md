# Mahatma Gandhi Biography Website

This repository contains a simple static website about Mahatma Gandhi – the Father of the Nation.  

## Features
- Biography and life story
- Early life, South Africa journey, and role in India’s freedom movement
- Principles of truth and non-violence
- Photo gallery section
- Styled using external CSS

## Usage
Open `index.html` in a browser to view the website locally, or visit the live version via GitHub Pages:

[Live Demo](https://username.github.io/gandhi-website/)

## Files
- `index.html` – Main HTML page
- `style.css` – External CSS
- `images/` – Photo gallery images
- `.gitignore` – To ignore unnecessary files

## Mission & Vision
- **Mission:** Educate and inspire people by sharing Gandhi’s life story, values, and philosophy.
- **Vision:** Promote non-violence, truth, and peace in today’s world.
